# User Guide

Step-by-step walkthroughs for everyday tasks on 91 IDAM.

## 1. Sign up & log in

1. Click **Login** in the top-right of the header.
2. New user? Click **Sign up** in the modal — enter name, email and password.
3. A 6-digit OTP is emailed to you. Enter it to verify the account.
4. After verification you're auto-logged in. Future logins use email + password and OTP.

> Locked out after 5 failed OTPs? The account is frozen for 24 hours and you'll receive a lockout email.

## 2. Search properties

The fastest path is the **hero search bar** on the home page. Type a location, project name or keyword — suggestions appear after 2 characters.

For full filters, click **Properties** in the header (or press *View all* in the suggestion list).

| Filter | Options |
|---|---|
| Budget | 9 preset ranges (Under 5 Lac → Above 5 Cr) |
| Property type | Residential / Commercial |
| Area (sq.ft) | Numeric range |
| Posted by | Owner / Builder / Dealer |
| Localities | Free text |
| Approved by | DTCP / RERA / CMDA |
| Ownership | Freehold / DTCP / CMDA / Patta / Leasehold |
| Facing | N / S / E / W and combinations |
| Road width | Numeric range |

Sort with the dropdown above results: Newest, Price (low/high), Area (small/large), Title (A-Z / Z-A).

## 3. Open a property's details

Click any card in the search results. The detail page has:

- **Image gallery** — full-screen on click
- **Summary** — price, BHK, area, type, ownership, status
- **Map** — Google Maps URL is stored verbatim and opens in a new tab
- **Contact** — opens the enquiry popup (login required)

## 4. Contact an owner / agent

1. Click **Contact** on a property card or detail page.
2. Login is required. Use the modal that appears.
3. The enquiry is recorded against the property and surfaced to the owner under **Profile → View Responses**.

## 5. List a property

You must be logged in. Two entry points:

- Header → Profile → "Add new property" (in My Listings section)
- Direct: `/start-selling`

Required fields:

- Title
- Property type (Residential / Commercial) — drives later category options
- Location: area name, city, pincode (Google Maps URL is optional, stored as-is)
- Selling price + area (sq.ft)
- At least one image (max 10, JPG/PNG, ≤5 MB each)

Submit goes to the super-admin for approval. You're notified by email once published.

## 6. Manage your dashboard

The user profile (`/user-profile`) has a sidebar with these sections:

| Section | What it does |
|---|---|
| Profile | Update name, email, phone, profile picture |
| Change password | Old + new password |
| My Listings | Your posted properties — edit, delete, view enquiries |
| View Responses | Enquiries grouped by property — approve / reject |
| Contacted | Agents you've contacted |
| Manage *(super-admin only)* | Review pending agents and properties |
| Requests *(super-admin only)* | Pending registration requests |

## 7. Browse agents

Header → **Our Agents** → grid of all verified agents. Click any card to open the full profile (photo, listings, contact details, ratings).

## 8. Become a partner

Header → **Individuals** or **Franchise Partners** dropdown → **Form Registration** opens a modal capturing your interest. The lead is forwarded to the 91 IDAM partnerships team via Zoho CRM.

## 9. Take the in-app tour

Look for the floating **"Take a tour"** button (bottom-right) — it appears on every page that has a guided walkthrough configured. The home tour also auto-starts on your first visit.
